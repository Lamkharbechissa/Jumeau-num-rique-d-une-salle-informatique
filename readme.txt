------------
Digital Twin Project Startup Guide

Requirements
- Node.js and npm installed
- PHP installed

Step 0: Extract files

Step 1: Frontend (digitaltwin folder)
Start a terminal and run:
cd digitaltwin
cd react
npm run dev

Step 2: Main Backend (projet digitaltwin folder)
Start another terminal and run:
cd "projet digitaltwin"
php artisan serve

## Step 3: Lamp Service (mini projet -lamps folder)
Start another terminal and run:
cd "mini projet -lamps"
php artisan serve
